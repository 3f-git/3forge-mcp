# AMI Web — Client-Side Patterns

## When to Activate

Activate when writing client-side AMI Web code involving:
- Panel callbacks and event handlers
- DataModel subscriptions
- Session and layout manipulation
- Real-time query execution in Web context
- Layout lifecycle and startup behavior
- Designing or implementing TablePanel, ChartPanel, HeatmapPanel, or HTML panels
